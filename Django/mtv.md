# MTV 패턴

- 코딩 공부 시 이해가 가지 않는 순간이 오는 이유

1. 개발은 혼자 하는 것이 아니다
2. 보안상의 이유
3. 개발자는 반드시 실수를 한다

   -> MTV패턴 사용하는 이유

- 개발 파트

  - Front End
    - HTML
    - CSS
  - Back End

    - 데이터처리

    -> 장고는 이보다 조금 더 세분화 해서 3파트(MTV)로 나눔

## MTV패턴

- Model(be)
  - 데이터베이스(DB)
- Template(fe)
  - 사용자가 보이는 영역
  - HTML
  - CSS
  - 탬플릿 언어
- View(be)
  - 데이터를 처리하는 곳
  - MTV중에서 핵심
